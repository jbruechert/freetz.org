packages/htpdate
================
htpdate
=======

Dieses Paket kann dazu genutzt werden die Uhrzeit der Fritzbox zu
synchronisieren. Dazu wird das HTTP Protokoll genutzt und dessen Header
ausgewertet. Eine Synchronisierung über Proxyserver ist möglich.
